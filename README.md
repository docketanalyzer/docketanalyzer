### Developer Setup

````
git clone https://github.com/docketanalyzer/docketanalyzer
cd docketanalyzer
pip install -e .
da dev install-extension chat
da dev install-extension core
da dev install-extension ocr
da dev install-extension pipelines
da configure
````
